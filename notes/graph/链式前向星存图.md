### 链式前向星存图

链式前向星存图是比赛中常用的存图方式, 与 邻接矩阵, 邻接表相比, 链式前向星能够使用更少的空间, 具体来说, 如果节点数为`n`, 边数为`m`, 那么链式前向星可以使用`O(m + n)`的空间保存这个图

首先假设我们要存的图是有向图, 在最后我会将存图方式推广到无向图

链式前向星存图主要依赖于三个一维数组: `head[]`, `next[]`, `to[]`, 接下来我会介绍这三个数组的含义

1. `head[]`: `head[i]`表示起始节点为`i`的第一条边的下标
2. `next[]`: `next[i]`表示和下标为`i`的边具有相同的起始节点的下一条边的下标
3. `to[]`: `to[i]`表示下标为`i`的边的结束节点的下标

> 再强调一遍, 这里我们说的表示方式, 都是针对有向图, 对于无向图的情况, 会在最后介绍

需要注意的是, 一般情况下, 节点的编号都会在题目中给出, 但是这里还会用到 边的编号 , 这个信息在题目中一般不会给出, 因此这里我们需要在建图的时候维护一个全局变量`int cnt = 1` (为什么初始值是`1`下面会提到), 每次插入一条新的边, `cnt++`, 这样我们就可以让`cnt`作为图中边的编号

> 上述使用全局变量`cnt`的思想, 在`Trie`的静态数组实现方式中也用到过

在介绍具体的有关链式前向星存图的遍历过程之前, 我想简单的概括一下链式前向星保存信息的逻辑, 这点对于理解链式前向星的存图方式, 以及理解如何遍历链式前向星存的图都至关重要

`head[i]`保存的是起始节点为`i`的第一条边的下标, 通过`head[i]`我们就可以找到从节点`i`出发的第一条边

那么接下来如何找从`i`节点出发的后续的边呢? 这里就用到了`next[]`数组, 由于`next[]`数组保存的是和当前边具有共同起始节点的下一条边的下标, 因此对于`i`节点来说, `next[head[i]]`就找到了从`i`节点出发的第二条边的下标, 以此类推, 从`next[]`数组中继续跳, 就能够找到从下标为`i`的节点出发的所有边的下标

找到了一条边的起始节点, 以及找到了这条边, 那么怎么找这条边的结束节点呢? 这时候就需要用到`to[]`数组了, `to[j]`表示的就是下标为`j`的边的结束节点的编号

概括来说, 链式前向星的存图逻辑就是: 点下标 --> 点相连的边 --> 边指向的点

接下来解释为什么用作边的下标的全局变量`cnt`要从1开始?

回顾`next[]`数组的定义可以发现, `next[]`数组的定义其实有点类似静态链表, 即 将以某一个点为起始节点的这些边使用类似链表的方式串联起来

那么如何判断一条链表的结束呢? 这里由于边的下标是从`1`开始计数, `0`这个下标没有用, 因此我们可以规定`next[i] == 0`表示和下标为`i`的边有共同的起始节点的边已经都遍历完了

还有一个问题, 如果要想插入一条新的边, 那么此时需要怎么表示?

首先使用`cnt++`分配一个边的下标, 然后将这条边插入到`next[]`数组中, 假设这条边的起始节点下标为`j`, 那么我们此时就需要将下标为`cnt`的这条边插入到`next[]`中以`j`下标的节点为起始节点的这些边的链表中, 显然为了插入的时间复杂度尽可能的低, 我们应该使用头插法来插入, 使用头插法之后, 我们需要做这样的更新: `next[cnt] = head[j], head[j] = cnt`, 并且更新`to[cnt]`即可

如果这个有向图的带有边权, 那么应该怎么处理? 

很简单, 只需要再添加一个`weight[]`数组即可, 因为在建图的过程中, 我们有了每一条边的编号, 因此`weight[i]`就可以表示下标为`i`的边的权重是多少

如何推广到无向图?

终于来到了最后一个问题, 链式前向星如何保存无向图?

对于无向图中的每一条无向边, 我们都可以拆分成两条有向边, 然后分别添加即可

链式前向星存图的部分代码: 

```java
class Graph{
    private int[] head;
    private int[] next;
    private int[] to;
    private int[] weight;
    private int cnt;
    // n: 节点数, m: 边数
    Graph(int n, int m){
        head = new int[n];
        next = new int[m];
        to = new int[m];
        weight = new int[m];
        cnt = 1;
    }

    // 添加一条 a -> b 的有向边
    public void add(int a, int b){
        next[cnt] = head[a];
        head[a] = cnt;
        to[cnt] = b;
        cnt++;
    }

    // 遍历节点a的所有邻接点

}
```
