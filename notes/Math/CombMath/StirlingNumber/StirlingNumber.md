斯特林数

第二类斯特林数

一、第二类斯特林数的定义
S(n,m)表示将n个不同的元素划分为m个非空且无标号的集合的方式数目。
例如：S(3,2)=3，因为将 3 个元素分为 2 个非空集合的方有：
{A,B}, {C}  
{A,C}, {B}  
{B,C}, {A}
这些集合之间没有顺序，例如 {A,B}, {C} 和 {C}, {A,B} 视为同一种划分方式。

二、斯特林数的递推关系
第二类斯特林数的递推公式为：S(n,m)=S(n−1,m−1)+m*S(n−1,m)
解释：
    1. 新增一个集合：第n个元素单独成为一个新集合，此时需要前n-1个元素已分成m−1个集合，对应S(n−1,m−1)。
    2. 加入现有集合：第n个元素放入已有的m个集合中的任意一个，共有m种选择，对应m*S(n−1,m)

三 计算
斯特林数显然可以依据上面的递推关系通过dp进行计算: 
```java
// 计算dp[n][m]
// dp[i][j]: 将[0, i]区间的元素分成[0, j]区间的集合, 此时的方案数
int[][] dp = new int[n + 1][m + 1];     // 初始化添加了辅助节点
dp[0][0] = 1;
for(int i = 1;i <= n;i++){
    for(int j = 1;j <= m;j++){
        dp[i][j] = dp[i - 1][j - 1] + dp[i - 1][j] * j;
    }
}
return dp[n][m];
```
