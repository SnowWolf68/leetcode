§1.1 基础

长度确定为k：



```cc
// 看作是可变窗口的特例
int n = s.size(), ret = 0, cur = 处理第一个元素, r = 0;
for (int i = 0; i < n; i++) {
    while (r + 1 < n && r - i + 1 < k) {
        r++;
        处理s[r]
    }
    if(r - i + 1 == k){
        更新ret
    }
    解除cur[i]的影响;
}
```
