dp总结

Motivation: 虽然做了很多的dp, 但是过段时间看看之前的问题还是没有思路, 因此决定对dp问题做一个总结, 总结一些自己的东西出来

PART 1: 背包dp

[LC871](https://leetcode.cn/problems/minimum-number-of-refueling-stops/description/)

在我们走到target的过程中, 会走过沿途的所有加油站, 对于每一个加油站, 此时我们都有两种选择, 1)加油 2)不加油  又由于背包dp的核心是 "选或不选" , 那么我们可以尝试用背包dp来解决这个问题

1. 考虑状态定义: 首先根据背包dp的要求, 肯定要有一个维度`i`来表示当前考虑`[0, i]`区间的元素, 除此之外, 由于本题问的是 "最低加油次数" , 那么显然还需要一个维度`j`来表示 "当前加了多少次油" , 最后是`dp[i][j]`的值的含义, 我们可以将其定义为能够走到的最远距离, 因此最终的状态表示我们定义为:  `dp[i][j]`表示考虑下标在`[0, i]`区间的加油站, 加了`j`次油, **能够走到的最远距离**

2. 接着考虑状态转移方程: 对于下标为`i`的加油站, 此时我们有两种选择 (这也是背包dp的经典套路): 加油 或 不加油

   1. 加油: 要想在下标为`i`的加油站加油, 首先需要满足条件: 能够走到下标为`i`的加油站, 即满足 

      `dp[i - 1][j - 1] >= stations[i][0]`
   
      如果能够满足, 那么我们就可以在`dp[i - 1][j - 1]`的基础上多跑`stations[i][1]`这么远, 即
   
      `dp[i][j] = dp[i - 1][j - 1] + stations[i][1];`
   
   2. 不加油: 此时问题转化为: 考虑`[0, i - 1]`区间的加油站, 加`j`次油, 能够走到的最远距离, 即此时有
   
      `dp[i][j] = dp[i - 1][j];`

​	对于上面两种情况, 我们此时显然需要取一个最大值(max)

3. 考虑初始化: 由于这里`i - 1, j - 1`都有可能越界, 因此需要考虑初始化问题. 一般来说这里都是选择添加一行一列辅助节点, 但是由于这里列表示: 加了`j`次油. 因此显然添加的一列辅助节点没有实际含义, 因此最终我选择只添加一行辅助节点, 其中第一行表示 不考虑任何的加油站

​	如何初始化? 由于第一行表示不考虑任何加油站, 那么除了`dp[0][0]`这个位置, 其余位置都是非法, 因为不考虑任何加油站的情况下只能加`0`次油, 因此初始化第一行(除去`dp[0][0]`)为`-INF`, 第一列表示加了`0`次油, 因此初始化第一列为`startFuel`

> 有关这里非法位置到底是初始化为`0`还是初始化为`-INF`的问题: 
>
> 结论: 都可以
>
> 解释: 假设这里我们初始化成`0`, 那么就会导致后面的某些状态, 可能会依赖到这里的"不考虑任何加油站, 并且加了`j (j > 0)`次油"的状态, 但是即使依赖于这个状态, 也不会导致后续更新的错误. 因为如果依赖了这个状态, 那么意味着其相比于依赖 `dp[0][0]` (即不考虑任何加油站, 并且加了`0`次油)  的状态少加了若干次油, 那么显然其能够走到的最远距离就要比其依赖于 `dp[0][0]` 要近, 那么这种依赖情况显然不会是最优解, 因此不会影响整个dp表的正确更新

4. 考虑填表顺序: 从上往下, 从左到右填表即可
5. 返回值: 本题问的是到达目的地的最少加油次数, 那么我们从小到大可以遍历`dp`表中的`dp[n][i]`这些列, 找到第一个满足`dp[n][i] >= target`的`i`, 即为最少加油次数

需要注意的是, 这里在添加了一行辅助节点的情况下, 状态转移中的下标需要作一下变换: 

`dp[i - 1][j - 1] >= stations[i][0] --> dp[i - 1][j - 1] >= stations[i - 1][0]`

`dp[i][j] = dp[i - 1][j - 1] + stations[i][1] --> dp[i][j] = dp[i - 1][j - 1] + stations[i - 1][1]`

```java
public int minRefuelStops(int target, int startFuel, int[][] stations) {
      int n = stations.length, INF = 0x3f3f3f3f;
      int[][] dp = new int[n + 1][n + 1];
      Arrays.fill(dp[0], -INF);		// 不写这行也可以
      for(int i = 0;i <= n;i++) dp[i][0] = startFuel;
      for(int i = 1;i <= n;i++){
          for(int j = 1;j <= n;j++){
              dp[i][j] = dp[i - 1][j];
              if(dp[i - 1][j - 1] >= stations[i - 1][0]){
                  dp[i][j] = Math.max(dp[i][j], dp[i - 1][j - 1] + stations[i - 1][1]);
              }
          }
      }
      for(int i = 0;i <= n;i++){
          if(dp[n][i] >= target) return i;
      }
      return -1;
  }
```
