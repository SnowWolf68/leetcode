恰好装满 & 至少 两种类型 0-1 背包的区别

[题目链接](https://leetcode.cn/problems/ones-and-zeroes/)

**恰好装满:** 

​	`dp[i][j][k]`: 考虑`[0, i]`区间的元素, 有`j`个`0`, `k`个`1` 的子集的最长长度

​	状态转移方程: 

​		前提: `j - list.get(i - 1)[0] >= 0 && k = list.get(i - 1)[1] >= 0`

​		转移: `dp[i][j][k] = Math.max(dp[i][j][k], dp[i - 1][j - list.get(i - 1)[0]][k - list.get(i - 1)[1]] + 1);`

​	初始化: `dp[0][0][0] = 0, dp[0][j][k] = -INF`

​	返回值: `return max(dp[sz][j][k]);`

**至少:** 

​	`dp[i][j][k]`: 考虑`[0, i]`区间的元素, 不超过`j`个`0`, `k`个`1` 的子集的最大长度

​	转移(**不变**): `dp[i][j][k] = Math.max(dp[i][j][k], dp[i - 1][j - list.get(i - 1)[0]][k - list.get(i - 1)[1]] + 1);`

​	初始化(**变**): `dp[0][j][k] = 0`

​	返回值(**变**): `return dp[sz][m][n];`



> 总结: 只有 **初始化** 和 **返回值** 有区别, 状态转移方程保持不变
