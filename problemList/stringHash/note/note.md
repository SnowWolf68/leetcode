# 字符串哈希

## 简介

字符串哈希就是通过某种方法, 求出某个字符串的哈希值, 并利用这个哈希值来快速比较两个字符串是否相等

## 主要思想

字符串哈希的主要思想有两点: 

1. 哈希函数的选取
2. mod值的选取

### 哈希函数的选取

一般来说, 我们通常使用 多项式函数 作为字符串的哈希函数

这里的多项式函数指的是: 

对于`str = abcde`, 其所对应的多项式函数为

`f = a * p ^ 4 + b * p ^ 3 + c * p ^ 2 + d * p ^ 1 + e * p ^ 0`

这个多项式函数可以理解为 p进制 表示, 其中`a, b, ...`就相当于是某个数位上的数, 而`p`就相当于是`p`进制中的基数

这里的`p`我们一般选取较大的质数

### mod值的选取

因为哈希函数`f`的值域较大, 为了能够将所有的哈希值映射到有限的空间中, 这里采用的是取模的方式, 但是需要注意的是, 因为这里对哈希值进行了取模, 因此就会出现哈希冲突, 因此mod值的选取和哈希冲突出现的概率是相关的

### 计算

情景: 给定一个字符串`str`, 要求计算出`str`中所有子串的哈希值

字符串哈希中有两个非常重要的数组: `h[]`数组和`p[]`数组

其中`h[i]`表示`[0, i]`这个前缀的哈希值, 之所以用前缀, 是因为后续可以利用类似前缀和的思想, 计算出所有子串的哈希值

`p[i]`表示`str`用`p`进制展开之后, 展开式各项的权值(比如`p[0] = p^0, p[1] = p^1, p[2] = p^2`)

#### 计算`h[]`数组和`p[]`数组

##### `h[]`数组

`str`使用`p`进制的展开式如下: (这里假设`str`的长度为5)

`str[0] * p^4 + str[1] * p^3 + str[2] * p^2 + str[3] * p^1 + str[4] * p^0`

这个展开式其实也就是`str[0, 4]`区间的子串(其实就是`str`本身)的哈希函数, 即

> 假设这里我们使用`str[i, j]`表示`str`位于`[i, j]`区间的子串对应的哈希值

`str[0, 4] = str[0] * p^4 + str[1] * p^3 + str[2] * p^2 + str[3] * p^1 + str[4] * p^0`

通过上面`h[]`数组的定义我们可以得到, `h[4] = str[0, 4]`, 那么有

`h[4] = str[0] * p^4 + str[1] * p^3 + str[2] * p^2 + str[3] * p^1 + str[4] * p^0`

以此类推, 我们能够得到`h[4]`的值

`h[4] = str[0] * p^3 + str[1] * p^2 + str[2] * p^1 + str[3] * p^0`

因此我们能够得到如下的递推式: `h[i] = h[i - 1] * p + str[i]`

通过这个递推式, 我们就能够利用类似DP的方法计算出`h[]`数组的所有值

需要注意的是, 利用DP来求这个递推式的时候, 其中的`i - 1`有可能会越界, 因此从DP的角度来理解的话, 我们需要给`h[]`这个DP表添加一个辅助节点`h[0]`, 并且初始化`h[0] = 0`, 这样添加了辅助节点之后, `h[]`数组的定义就发生了一些变化: `h[i]`表示的是**前i个字符**组成的子串的哈希值, 换句话说, `h[i]`表示的是`str`在`[1, i]`区间的子串的哈希值(即下标变成了从1开始)

##### `p[]`数组

`p[]`数组的计算就比较简单, 由于我们知道`p[i] = p^i`, 因此只需要简单的递推(也可以理解为DP)计算即可

#### 如何利用`h[]`数组和`p[]`数组得到`str`每一个子串的哈希值

这一点非常关键, 之前的`h[]`数组只是计算了`str`中每一个前缀`[1, i]`的哈希值, 还没有得到`str`中每一个子串的哈希值, 这里我们就需要利用`h[]`数组和`p[]`数组, 计算出`str`中每一个子串的哈希值

可以举个例子: 

`h[5] = str[0, 4] = str[0] * p^4 + str[1] * p^3 + str[2] * p^2 + str[3] * p^1 + str[4] * p^0`

`str[2, 4] = str[2] * p^2 + str[3] * p^1 + str[4] * p^0`

那么通过`h[5]`(`str[0, 4]`)如何计算出`str[2, 4]`?

通过观察上面两个式子可以得到: 

`str[2, 4] = (str[0] * p^4 + str[1] * p^3 + str[2] * p^2 + str[3] * p^1 + str[4] * p^0) - (str[0] * p^4 + str[1] * p^3)`

`str[2, 4] = (str[0] * p^4 + str[1] * p^3 + str[2] * p^2 + str[3] * p^1 + str[4] * p^0) - (str[0] * p^1 + str[1] * p^0) * p^3`


即`str[2, 4] = str[0, 4] - str[0, 1] * p^(4 - 2 + 1)`

更一般的, 有`str[i, j] = str[0, j] - str[0, i - 1] * p^(j - i + 1)`

其中, `p^(j - i + 1)`即为`p[j - i + 1]`, 并且`str[0, i] = h[i + 1]`, 因此最终的式子为

`str[i, j] = h[j + 1] - h[i] * p[j - i + 1]`

因此我们就得到了`str`中任一子串的哈希值

#### 何时MOD? 

由于这里我们有取模的需要, 因此只需要在上面每一步的计算中都进行取模即可, 具体来说: 

`p[i] = (p[i - 1] * p) % MOD`

`h[i] = ((h[i - 1] * p) % MOD + str[i - 1]) % MOD`

`str[i, j] = ((h[j + 1] - (h[i] * p[j - i + 1]) % MOD) + MOD) % MOD`

### Template

```
public class StringHashBasic {
    private int p;
    private int MOD;

    private int n;          // 字符串长度
    private int[] pre;      // 哈希值前缀
    private int[] pow;      // pow[i] = p^i

    StringHashBasic (String str, int p, int mod) {
        this.p = p;
        this.MOD = mod;
        this.n = str.length();

        pre = new int[n + 1];
        pow = new int[n];

        pre[0] = 0;
        pow[0] = 1;

        for(int i = 1;i < n;i++){
            pow[i] = (pow[i - 1] * p) % MOD;
        }

        for(int i = 1;i <= n;i++){
            pre[i] = ((pre[i - 1] * p) % MOD + str.charAt(i - 1)) % MOD;
        }
    }

    int getHashValue(int l, int r){
        return ((pre[r + 1] - (pre[l] * pow[r - l + 1]) % MOD) + MOD) % MOD;
    }
}
```
